## 不可变对象
>一旦创建就不可修改的对象：字符串、数字、元组。

1. 不可变对象地址值不变；
2. 讨论深浅拷贝无意义，深浅拷贝之后地址不变。

<!-- more -->

## 可变对象
>可以修改的对象：列表、字典。

一般来说深浅拷贝是针对可变对象讨论的。

## 浅拷贝
>只拷贝了顶级对象本身，该对象引用的嵌套对象只拷贝其引用。

1. 切片
切片可用于字典外的所有序列：列表、字符串、元组
字典只能用D.copy()或D.deepcopy()

2. copy模块
import copy
x = copy.copy(y)

## 深拷贝
>拷贝所有对象，包括顶级对象本身以及该对象引用的嵌套对象。

copy模块
import copy
x = copy.deepcopy(y)

注：copy模块可应用于序列（列表、元组、字符串），以及字典。
